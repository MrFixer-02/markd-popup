# /markd.

Save any web page to your personal library — organized, searchable, synced. No setup.

**You:** download → load in your browser → sign in with email → start saving.
**We** handle the cloud, so your saves follow you everywhere. Nothing to install, configure, or maintain.

---

## Install — Chrome

1. Download **[markd-chrome.zip](downloads/markd-chrome.zip)**
2. Unzip to a folder (e.g. `markd-chrome`)
3. Open `chrome://extensions`
4. Turn on **Developer mode** (top right)
5. Click **Load unpacked** → choose the unzipped folder
6. Pin **/markd.** to your toolbar

> Works the same in **Edge**, **Brave**, and other Chromium browsers — use the Chrome ZIP.

## Install — Firefox

1. Download **[markd-firefox.zip](downloads/markd-firefox.zip)**
2. Unzip
3. Open `about:debugging#/runtime/this-firefox`
4. Click **Load Temporary Add-on…** → pick `manifest.json` inside the folder

> Temporary add-ons unload when Firefox restarts. A permanent listing may come later.

---

## Sign in

1. Click the **/markd.** icon
2. Tap the **person** icon (Account)
3. Enter your email → **Send code**
4. Paste the **6-digit code** from your inbox → **Verify**

Sign in with the same email on any computer and your whole library is right there.

## Quick use

| Action | How |
|--------|-----|
| Save the current page | **Capture** in the popup, or **⌥ + M** (Mac) / **Alt + M** (Windows) |
| Browse your saves | **Recents · Web · Folders** tabs |
| Make a folder | **Folders** tab → **+** |
| Move a save into a folder | Right-click a save → **Move to** |
| Share | Right-click a save or folder → **Share** → send the code |
| Import a share | Paste the code in the box → **↵** |

## What you can actually do with it

Saving a page is the easy part. **Folders** are where /markd. earns its place — group related pages so you never have to dig for them again.

- **Collect scattered posts.** Found a great LinkedIn post, a thread, a tip? You can't bookmark your way back through an endless feed. Capture it into a **“LinkedIn”** folder and every post you saved is in one place.
- **Job hunt.** Save every job posting into a **“Jobs”** folder, then **share the whole folder** with a mentor or friend with one code.
- **Research a topic.** Pull articles, docs, and references for a project into one folder — your own little reading desk.
- **Plan a trip.** Hotels, things to do, restaurant pages → a **“Trip to Tokyo”** folder you can share with whoever you're travelling with.
- **Recipes.** Save them from any site into **“Recipes”** instead of 30 open tabs you’ll never reopen.
- **Shopping / gift ideas.** Drop product pages into a **“Wishlist”** folder; share it so people know what to get you.
- **Study group.** Build a folder of tutorials and docs, share one code, and everyone imports the same set instantly.
- **Read later.** Long reads go into **“Read later”** — open them on your phone, laptop, anywhere you’re signed in.

The pattern is always the same: **save → drop into a folder → find it instantly later → share the whole set when you want.**
